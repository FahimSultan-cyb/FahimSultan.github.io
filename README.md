# Fahim Sultan — Researcher Portfolio (Demo)
This project is a modern, accessible researcher portfolio tailored from the provided CV and image.
- Auto theme by visitor timezone (day/night) + manual toggle (☾ / ☼).
- Color-blind friendly palette and accessible font (Inter).
- Publications loaded from `publications.json` (sourced from uploaded CV). If you want real-time Google Scholar sync, provide API or allow scraping.
- CV is included as `Fahim_CV.pdf`.

To run:
Open `index.html` in a web browser. This is a static site.

Notes:
- Some fields were populated from uploaded CV. Any missing items remain blank for you to fill.
